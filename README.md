```markdown
# AFAA PILATES 模擬試験 — Interactive Quiz

このリポジトリには日本語（AFAA PILATES 模擬試験 10問）をサンプルにした簡易的なクイズWebアプリが含まれています。

使い方
1. 上記ファイルを同じフォルダに保存します（index.html, styles.css, app.js）。
2. ブラウザで index.html を開いてください。
3. 「サンプル（AFAA PILATES）を読み込む」を押すと、模擬試験が読み込まれます。

機能
- 複数選択（正答が複数ある場合）に対応
- 回答の即時採点とフィードバック
- ローカルストレージへの進捗保存・復元
- 結果のエクスポート（JSON）
- ファイル/テキストからのクイズ読み込み

備考
- 出題順をランダム化したい場合は、app.js の読み込み処理にシャッフルを追加できます。追加希望があれば対応します。
```
